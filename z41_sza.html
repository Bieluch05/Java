Petla for:
konstrukcja:for (zainicjowanie_zmiennej; warunek_konczacy_wykonywanie_petli; zmiana_zmiennej) {kod ktory zostanie wykonany pewna ilosc razy}

Petla while z kontrolowanym wejsciem:
konstrukcja:while (wyrazenie_sprawdzajace_zakonczenie_petli) { ...fragment kodu ktory bedzie powtarzany... }

Petla do while z kontrolowanym wyjsciem:
konstrukcja:Do { ...fragment kodu ktory bedzie powtarzany... } while (false)
